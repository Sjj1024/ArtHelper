# 文章同步助手

可以帮助你同步你的csdn文章到其他平台，如博客园、掘金、知乎、简书等

## 同步文章的时候，有个状态说明  

有个状态说明或者弹窗提示，说正在同步，或者同步完成

## 功能

-   支持同步文章到多个平台
-   支持自定义文章标题、摘要、分类、标签
-   支持自定义文章封面
-   支持文章缓存
-   支持屏蔽csdn等平台广告
-   支持自定义同步规则
-   auto replace image url
